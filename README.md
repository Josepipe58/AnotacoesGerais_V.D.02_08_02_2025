# AnotacoesGerais_V.D.02_08_02_2025

# #🚀 Meu Projeto pessoal de anotações gerais.

Esse é um projeto de lembretes, configurações, códigos, etc.
Ele foi criado para desktop, com a tecnologia do WPF e a estrutura MVVM.
---

## 📖 Sobre

Este projeto foi desenvolvido para anotações em geral.  
Ele permite que se você pesquise quando esquecer de uma configuração, uma senha de um site, um trecho de código de programação ou qualquer lembrete que você esqueceu.
---

# #🛠 Instalação

Clone o repositório e instale as dependências:
Microsoft.EntityFrameworkCore.SqlServer(10.0.7)
Microsoft.Xaml.Behaviors.Wpf(1.1.142)
